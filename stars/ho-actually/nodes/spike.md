spike (design)

<!-- depth:2 -->
<!-- provenance: extracted -->
A purpose-built proof that a design's settled values hold up against real code constraints, made as the reference the production build copies from.

<!-- depth:3 -->
<!-- provenance: extracted -->
A hand-built artifact validating the frozen Basis of Design against the
real algorithmic constraints before production code; the reference target the implementation
ports rather than copies. A/B spikes test a second design variant as a toggleable
alternative.
