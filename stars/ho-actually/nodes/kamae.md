Kamae (構え)

<!-- depth:2 -->
<!-- provenance: extracted -->
The ready stance and all the framing done before a build, producing a chain of documents that move from opinions to a session plan.

<!-- depth:3 -->
<!-- provenance: extracted -->
The ready stance: everything that happens before (and framing around) the
build. Produces the five-document chain — seed (K1), system design (K2), README (K3), ho
overview (K4), per-ho documents (K5) — each step increasing commitment: opinions → decisions
→ scope → sequence → session.
